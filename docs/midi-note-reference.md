# MIDI Note Reference

Quick lookup table for MIDI note numbers, note names, and frequencies.

**Formula:** `frequency = 440 × 2^((note - 69) / 12)`

**Key landmarks:**
- Note 36 (C2, 65.4 Hz) — lowest note on most organ manuals
- Note 60 (C4, 261.6 Hz) — middle C
- Note 69 (A4, 440.0 Hz) — tuning reference
- Note 80 Hz crossover — approximately note 39–40 (D#2–E2)
- Note 96 (C7, 2093.0 Hz) — highest note on most organ manuals

**Organ keyboard ranges:**
- Manual (61 keys): notes 36–96 (C2–C7)
- Pedalboard (32 keys): notes 36–67 (C2–G4)
